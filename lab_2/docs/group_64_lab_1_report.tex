\subsection*{Worst Case of Approximation}
\addcontentsline{toc}{subsection}{Worst Case of Approximation}

In order to test the worst case for approx1() on a small graph of five nodes, you would have to calculate all possible graphs with 5 nodes. The maximum number of edges in a graph is expressed as:

\begin{align*}
    \text{max number of edges} &= \frac{n(n-1)}{2}\\
    &= \frac{5(5-1)}{2}\\
    &= \frac{20}{2}\\
    &= 10
\end{align*}

Therefore, a graph with five nodes can have at most ten edges. Furthermore, within a graph an edge can either exist or not exist. This means that we can express the total number of graphs as $2^{10}$, as for every max number of edges in the, there exist two options for an edge's existance. This means that there are $1024$ total possible graphs of size ten. In order to test for this, you can enumerate each subset of edges in a graph. 

\section*{Independent Set Problem}
\addcontentsline{toc}{subsection}{Independent Set Problem}

When understanding the basis of the Independent Set Problem, the relation between the Minimum Vertex Cover (MVC) and Maximum Independent Set (MIS) is extremely intuitive. The MIS is the largest set of nodes in a graph with edges that do not connect. This is effectively the opposite of the MVC, as the MVC is the smallest set of nodes in a graph that connects every to every edge. Subsequently, this would mean that set of nodes in MVC and the set of nodes in MIS converge such that $\text{MVC}\cup\text{MIS}=G$, where $G$ is the set of all nodes in the graph. Furthermore, in order to implement an MIS function, we can simply apply inverted logic to the already-existing MVC function.

\begin{verbatim}
def is_independent_set(G, C):
    for start in G.adj:
        for end in G.adj[start]:
            if start in C and end in C:
                return False
    return True

def MIS(G):
    nodes = [i for i in range(G.number_of_nodes())]
    subsets = power_set(nodes)
    maxSet = []
    for subset in subsets:
        if is_independent_set(G, subset):
            if len(subset) > len(maxSet):
                maxSet = subset
    return maxSet
\end{verbatim}
